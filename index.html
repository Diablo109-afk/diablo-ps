Windows / MacOS Guide
  • 💨 Press ⊞ Win+R, type: C:\Windows\System32\drivers\etc (Windows).
  • 📄 Find hosts file, right-click → Edit with Notepad (Admin).
  • 📋 Copy the Windows host lines using the button above.
  • ✏️ Paste them at the bottom of the hosts file, save.
  • 🖥️ For MacOS: Open Terminal → sudo nano /etc/hosts, paste entries, save.
